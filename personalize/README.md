# Personalize

Run a script on workspace start that allows developers to run custom commands to personalize their workspace.

```tf
module "personalize" {
  source   = "registry.coder.com/modules/personalize/coder"
  version  = "1.0.2"
  agent_id = coder_agent.example.id
}
```
